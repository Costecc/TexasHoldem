Detailed Description

Klasa Game sluzy do obslugi zdarzen zwiazanych z przebiegiem gry: tasowanie kart, przechowywanie talii, puli, rundy oraz zapisywanie i odczytywanie stanu gry z pliku czy eksportowanie historii rozdan po zakonczeniu gry.
